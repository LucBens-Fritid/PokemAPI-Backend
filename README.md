# PokemAPI-Backend

This application will let you search for any pokemon by name or id from the pokeApi

## Installation

Clone the project:
https://github.com/LucBens-Fritid/PokemAPI-Backend.git

Navigate to the project directory:

```bash
cd PokemAPI-Backend
```

Restore dependencies:

```bash
dotnet restore
```

Build the project:

```bash
dotnet build
```

Run the application:

```bash
dotnet run
```
